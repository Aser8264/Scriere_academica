# Analiza Algoritmilor de Sortare

Acest proiect conține un benchmark în C++ pentru compararea mai multor algoritmi de sortare pe seturi de date diferite.

Algoritmii implementați în versiunea curentă sunt:
- Bubble Sort
- Insertion Sort
- Selection Sort
- Merge Sort
- Quick Sort
- Heap Sort
- Shell Sort
- Radix Sort
- IntroSort (`std::sort`)

Documentația completă, în limba română, este disponibilă în [documentatie_sortare.md](documentatie_sortare.md).

Fișiere importante:
- [sort/main.cpp](sort/main.cpp) - implementarea principală a benchmark-ului
- [sort/results.csv](sort/results.csv) - rezultatele generate la rulare
